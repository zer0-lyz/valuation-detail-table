# 评估明细表字段定义总表

> 基于模板 `评估明细表-v1.90-FOR AI.xlsx` + 邦能达项目实战验证
> **用途**：Agent 填写评估明细表时的字段内容规范指引。每个字段明确"填什么、不填什么、怎么填"。
> **关联规则**：DT-46/DT-149/DT-172/DT-179（业务内容）、DT-30/DT-178（发生日期）、DT-111（结算对象）、DT-144（币种）、DT-168/DT-171（字段完整性）

---

## 一、字段分类体系

评估明细表的所有字段可分为 **6 类**：

| 类别 | 字段举例 | 核心规则 |
|------|---------|---------|
| **标识类** | 序号、编号、项目编号 | 连续整数，唯一不重复 |
| **文本描述类** | 结算对象、业务内容、开户银行、备注 | 禁止数值；按业务规则约束内容关键词 |
| **日期类** | 发生日期、到期日、启用日期 | datetime 类型，特定格式 |
| **数值类** | 账面价值、评估价值、增值额 | 数字格式 `#,##0.00`，来源：科目余额表 |
| **比率类** | 增值率%、利率% | 百分比格式，通常为公式自动计算 |
| **公式类** | 增值额、账龄、外币换算 | 模板公式自动计算，禁止写入硬值 |

## 二、通用字段定义

以下字段在多个 Sheet 中重复出现，统一定义。

### 序号（B列 / 2号列）

| 属性 | 定义 |
|------|------|
| col_map 键 | `seq` |
| 数据类型 | 整数 |
| 格式 | `0`（整数，不显示小数） |
| 内容规则 | 从 1 开始连续编号，按实际数据行递增。跳过零值行后序号连续（DT-215） |
| 禁止 | 重复编号、跳号、非数字 |

### 结算对象 / 户名 / 欠款单位名称

| 属性 | 定义 |
|------|------|
| col_map 键 | `settlement` |
| 数据类型 | 文本 |
| 列名变体 | 户名（结算对象）、欠款单位名称（结算对象）、收款单位名称（结算对象）、欠款单位（人）名称（结算对象）、开户银行、存放部门（单位）、名称及内容、项目及内容 |
| 内容规则 | 往来科目：按辅助余额表或科目余额表末级子科目展开（DT-111）。结算对象必须与科目余额表/辅助余额表一致 |
| 禁止 | 为空（DT-168 CRITICAL）、填汇总数代替明细行（DT-111） |

### 业务内容 / 结算内容 / 款项内容

| 属性 | 定义 |
|------|------|
| col_map 键 | `business` |
| 数据类型 | 文本 |
| 列名变体 | `业务内容`（标准往来科目）、`结算内容`（合同类/负债类科目）、`款项内容`（递延收益）——搜索 MUST 同时查 3 个变体（DT-179） |
| 内容规则 | **2-6 字业务关键词**，从序时账摘要归纳（DT-60）。优先从摘要高频关键词提取（货款/服务费/工程款/保证金等），无摘要时用 `infer_business_content()` 推断 |
| **禁止** | **填结算对象名称**（DT-172）、填科目名称（DT-149）、长摘要（>15字）、含"公司/有限"、空值（DT-171） |
| 关键词白名单 | `货款` `服务费` `工程款` `保证金` `检测服务费` `借款` `往来款` `报销款` `备用金` `租金` `税费` `社保` `设备款` `材料款` `保费` `运费` `预付款` `押金` |

### 发生日期

| 属性 | 定义 |
|------|------|
| col_map 键 | `date` |
| 数据类型 | datetime |
| 格式 | `yyyy"年"m"月"`（DT-30），COM resave 后需二次修复 |
| 内容规则 | 取距基准日最近的同方向凭证日期（DT-178）：借方余额科目取最近借方日期，贷方余额科目取最近贷方日期 |
| 禁止 | 字符串类型、为空（无序时账除外 DT-143）、填"无"/基准日等占位文本 |

### 账面价值 / 评估价值

| 属性 | 定义 |
|------|------|
| col_map 键 | `book_value` / `assessed_value` |
| 数据类型 | 数值 |
| 格式 | `#,##0.00` |
| 数值方向规则 | 按科目性质决定正负号：资产类借方余额=正数，贷方余额=负数；负债类贷方余额=正数，借方余额=负数 |
| 来源 | 科目余额表末级余额。优先辅助余额表按结算对象拆分，无辅助余额表时按末级子科目填列 |

### 币种 / 外币账面金额 / 评估基准日汇率

| 属性 | 定义 |
|------|------|
| col_map 键 | `currency` / `foreign_amt` / `exchange_rate` |
| 数据类型 | 文本 / 数值 / 数值 |
| 内容规则 | **无外币时全部留空**（DT-144），禁止填入"CNY"/"1.0"等默认值。有外币时币种填"人民币"/"美元"等 |

### 备注

| 属性 | 定义 |
|------|------|
| col_map 键 | `remark` |
| 数据类型 | 文本 |
| 内容规则 | 差异说明、减值原因、特殊事项。科目余额表与 BS 差异标注原因（留待补资料） |

### 增值额 / 增值率%

| 属性 | 定义 |
|------|------|
| col_map 键 | `increment` / `increment_rate` |
| 数据类型 | 公式 |
| 内容规则 | 增值额 = 评估价值 - 账面价值；增值率% = 增值额 / 账面价值 × 100。**禁止手动写入硬值** |

## 四、特殊字段规则

### 4.1 坏账准备行（DT-151↑ / DT-18）

| 行 | I列（账面价值） | J列（评估价值） |
|----|---------------|---------------|
| 坏账准备 | `+abs(坏账金额)` **正数** | `0` |
| 预计风险 | `0` | `+abs(减值评估值)` **正数** |

### 4.2 固定资产/无形资产备抵科目联动

| 备抵科目 | 公式 |
|---------|------|
| 账面净值 | = 账面原值 - 累计折旧/摊销 |
| 评估价值 | = 评估原值 - 评估贬值（通常不需显式计算） |

### 4.3 应交税费借方余额

应交税费科目若为借方余额（如留抵增值税），账面价值填负数，并在备注标注"借方余额"（DT-147）。

### 4.4 零值行跳过规则（DT-215）

- 普通科目：账面价值 = 0 则跳过整行
- 长期资产（contra）：账面原值 = 0 **且** 账面净值 = 0 才跳过
